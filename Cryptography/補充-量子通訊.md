# 量子不可克隆

不可能複製出一模一樣的量子。

# 量子的觀測不對異性

對於測量者，光子測量前的狀態未知。

光子有四種偏振狀態: 

1. 圓偏振
	* 左旋
	* 右旋

2. 線偏振
	* 水平
	* 豎直

且觀測的順序會影響觀測的結果，也就是說先觀測圓偏振取得的線偏振結果可能與先觀測線偏振取得的圓偏振結果相異。

# 通訊過程

假設現有兩人A和B進行通訊。假設A發送4顆光子給B，同時隨機使用不同的觀測方式觀測光子的偏振。

B收到光子後，亦同樣隨機使用不同觀測方式觀測光子的偏振。同時B會得知A在觀測哪幾個光子時使用的方法與自己相同，捨棄那些觀測方式不相同的光子數據，將觀測方法相同的光子數據做為密鑰加密數據。

# 如何知道有人監聽? 

假設有一人C在監聽A與B兩人的通訊，則取得光子後亦隨機使用不同觀測方式觀測光子的偏振，但由於C不知道A的觀測方法，所以隨機觀測後再轉發給B的光子被B測量後的結果可能與A不同，如果B在確認觀測方式與A相同時取得的結果與A相異，便可知道有人監聽，則丟棄這次傳輸。
