
# 什麼不是故事

- 沒有意義的事
- 平淡無奇的事
- 沒頭沒尾的事：必須要有因果的描述
- 求證真偽的事
- 爭辯對錯的事
- 需不斷解釋的事

# 故事的五要素

- 背景
- 人物 ＋ 事件
- 情節
- 主題
- 故事

# 故事的六要素

- 時間
- 空間
- 人物
- 原因
- 經過
- 結果

# 核心概念

主題：故事談論的內容。

主旨：故事談論的目的。

意涵：故事的含義。

核心概念：故事中能整合所有元素且具有意義的概念。
