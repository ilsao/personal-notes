
# 一月

時間悄然流逝，轉瞬間就迎來了新年。距離我正式離開華東也已經過去快要半年，學弟妹們也正準備著學測。

我回到母校幾趟，見了之前的導師，也如願以償的嘗到了新食堂的味道。不錯，還是和之前一樣普通。

上學期的總成績還是沒出，導致去年的年終總結總是不能發出。縱觀去年，學測後屬實是去了許多地方，與摯友們的回憶也正烙印在腦海中揮之不去。眼下他們即將放假，總算能見個面聊聊各自的生活趣事。

話雖如此，自我學習仍不可落下。讓我談談自暑假起我做的一些事吧！

首先，懷著自小學起做遊戲的夢想，總算認認真真的學起了 Unity。雖然至今只看了三分之一左右的內容，但仍學到了許多。畢竟整部教學時長約 60 小時呢！

緊接著遊戲開發的是圖形學，我開始看起了著名的 Games 101。至於官方作業，在 Windows 11 上虛擬機的配置有些許問題不支持運行。我還嘗試在本機配置 VS Code 開發，也遇到了許多不正確的結果。於是我想，我應該首先認真學習課程中的內容，而非浪費時間重複配置。所以打算學完 Games 101 所有內容後直接用 DirectX 12 或 OpenGL 重新實現所有課程內容。
我學到了：
- MVP 矩陣變換
- 光柵化
- 抗鋸齒
- Shadow Map
- 渲染方程

最後，我還參與了國科大一生一芯計畫，目前學習到預學習 E 階段。此階段結束就是正式提交入學申請了。在 F 階段，我使用 logisim 搭建了一個非流水線處理器，支持部分 RISC-V 指令，並搭配一個圖像顯示 I/O 映射地址區域。

具體地，在預學習 E 階段，我學到了：
- Verilog
- Vim 與 Tmux
- Linux 基礎指令
- Makefile 基礎
- 如何用 C 實現類
- 使用 C 實現 ISA emulator
- Verilator 基礎
- 了解 Verilog 的仿真行為
- 了解邏輯綜合的步驟
- 了解處理器後端設計流程
- 完成南京大學數字邏輯與計算機組成部分實驗

同時，軟體開發社的工作也開始了。我們當作舍教授的研究助理，協助開發科學教育 (SciEdu) 相關的網站。

其中，我擔任的是 SciEdu 的網站後端，使用 Go 語言開發。我們使用敏捷開發，並且通過 mockery 編寫 Table-Driven Test。在實際開始編寫代碼前，我們會繪製 ERD 與 ClassD。實踐時，我們使用三層架構 (Handler -> Service -> Querier)。

最後，在一月的盡頭，我開始重新學習機器學習。並且，決定延後學習 Unity 遊戲開發。因為這塊領域屬於應用實踐，我認為現在更應打牢基礎。

# 二月

經過一月末與二月初的學習，我重新學習了深度學習基礎與 pytoch 的使用。雖然如此，但仍停在較淺的階段：線性回歸、softmax 分類、多層感知機、Dropout、權重衰減等。話雖如此，我也完成了以下項目：
- 基於多層感知機的 Fashion-MNIST 訓練集識別
- Kaggle 房價預估，Score 0.12885 排名 1329 / 4251

我意識到機器學習過於注重數學推導，而非工程實踐 (當然可能與我挑的書 Dive Into Deep Learning 有關係，此書以嚴謹的推導著稱)，如果沒有目的推進我的學習，恐怕無法心甘情願的學下去。

於是，我快速的推遲了該領域的學習。轉向將之前沒完成的操作系統項目 BeyondOS 完成。

BeyondOS 是我在跟隨 OS in 1000 lines 寫出來的項目，我對教程中的部分代碼進行了改進 (大部分集中於 `syscall` 的處理)，並且將各部分代碼劃分成各個文件，避免所有代碼都擠在 `kernel.c` 中。

BeyondOS 包含以下功能：
- 進程管理：基礎的 cooperative multitasking
- 虛擬內存：每個進程擁有一個頁表，並且會隨著上下文切換而切換
- `syscall`：用戶與內核模式的切換
- ...

你可以在[這裡](https://github.com/ilsao/BeyondOS)找到我的項目。

隨著操作系統項目的完結，我開啟了另一個項目的學習：Ray Tracing in One Weekend。當然也因為現在 (2026/2/12) 快要開學了，剛好剩下一個多星期的時間讓我完成這個項目。

穿插在其間的，是閱讀好幾年前買下卻至今沒看過的書：程序員修練之道。我希望可以在開學前將這本書看完。

現在是 2026/2/14 情人節，我把 Ray Tracing in One Weekend 學完啦！

這是我的 repo：[https://github.com/ilsao/Ray-Tracing-Renderer](https://github.com/ilsao/Ray-Tracing-Renderer)。

同時，我還更新了我的個人網站。換了一個更學術風的模板，並把項目與自我介紹全部用英文表達。換了新的網站，舊的怎麼處理呢？我使用 project 的 github page 將其保留，並通過鏈接使二者互通。

歡迎你來我的網站看看：[https://ilsao.github.io/](https://ilsao.github.io/)。

我成功啦！在開學前看完了程序員修煉之道！

既然都學了 Ray Tracing in One Weekend 何不找真正的圖形學教程來看？所以，我開始去看 Games 202 高質量實時渲染，截止開學前，我學到了：
