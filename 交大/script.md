# 就讀校系

國立陽明交通大學
- 座標：新竹縣新竹市大學路 1001 號 (國立清華大學旁邊，由交清小徑完全連在一起)
- 飲食：去清華大學及其周邊商圈吃
- 網絡：曾經是全台網絡最快處，親測 $100\text{MB/s}$。
- 宿舍：新生宿舍剛翻新。
- 圖書館：書很多，適合來卷。而且還有漫畫屋，可以無限續借。還有十塊一杯的咖啡。
- 公共空間：有許多可供預訂的討論室，可以盡情討論課業
- 學長姐：進來有豐富的學長姐陪你聊天
- 體育：有免費健身房與游泳池與濃厚的跑步氛圍

資訊工程 (computer science)
- 威儀：loading 不重給分好看有上有分的物理課
- 大一上課程對標應數，但你包能過，可以取得濃厚的數學功底
- 適合人群
	- 做遊戲：以後想做遊戲的同學，讀這個科系包對  (圖形學或多媒體方向)
	- 當黑客：資安組有加分喔
- 沒國文，化學，生物。而且理科課不用上實驗課。
- 課表上的課較少，有充裕的自我學習時間
- 很多巨佬可以認識，拓展你的人脈
- 性別比相對平衡
- 可以加入軟體開發社，我們都加了
- 學長姐有豐厚的考古題與筆記，幫助考試
- 和藹可親且強的教授 (cmu cs 四大名校之一)
