Sol:
這題我的錯誤在於尋找 slant asymptote，所以以下僅講解如何尋找 slant asymptote。
因為我們知道 $x\to \infty$ 時函數有漸進線：$y=1$。所以想要驗證斜漸近線，則需要**考慮 $x\to -\infty$ 的情況**。
假設當 $x\to -\infty$ 時的斜漸近線為 $y=ax+b$。
$a=\lim_{ x \to -\infty } \frac{\frac{\sqrt{ e^{2x}+x^{2} }}{e^{x}-1}}{x}=\lim_{ x \to -\infty }\frac{\sqrt{ e^{2x}+x^{2} }}{xe^{x}-x}=\lim_{ x \to -\infty } \frac{-\sqrt{ \frac{e^{2x}}{x^{2}}+1 }}{e^{x}-1}=1$ 
(注意，因為 $x<0$，所以 $\sqrt{ e^{2x}+x^{2} }=|x|\sqrt{ \frac{e^{2x}}{x^{2}}+1 }=-x\sqrt{ \frac{e^{2x}}{x^{2}}+1 }$) (如果同除的是 $e^{x}$ 就不用加負號，因為 $x<0$ 時 $e^{x}>0$)
$b=\lim_{ x \to -\infty } \frac{\sqrt{ \frac{e^{2x}}+x^{2} }}{e^{x}-1}-x=\lim_{ x \to -\infty } \frac{-x\sqrt{ \frac{e^{2x}}{x^{2}}+1 }}{e^{x}-1}-x=0$ (注意，**這邊需要很複雜很複雜的夾擠才能得到**，考試就這樣寫吧，沒分就沒分了，總比沒時間寫後面來的好)
