這裡需要你去學一波**單調有界定理**。
簡單來說，一個函數如果在 $[a,b)$ 單調遞增且有上界，則 $\lim_{ x \to b }=\text{sup}\{f(x)|a\leq x<b\}$。反之，一個函數如果在 $[a,b)$ 單調遞減且有下界，則 $\lim_{ x \to b }f(x)=\text{inf}\{f(x)|a\leq x<b\}$。
用到這題：
$\lim_{ x \to c^{+} } \frac{f(x)-f(c)}{x-c}=\text{inf}\{f(t)|a\leq t<c\}$ (利用單調遞減)
$\lim_{ x \to c^{-} } \frac{f(x)-f(c)}{x-c}=\text{sup}\{f(t)|a\leq t<c\}$ (利用單調遞增)
不能保證上下確界相同，所以 $f'(c)$ 不一定存在。
又因為凹口向上需要確保：任意函數上二點的連線都在函數圖形的上方。
所以如果函數在任意一點不連續，都會破壞這個條件。所以，凹口向上的函數必定連續。
