# 物理史

![[Pasted image 20250909165841.png]]

四大古文明：巴比倫、埃及、印度、中國

柏拉圖的徒弟 => 亞里斯多德 => 亞歷山大的老師

亞里斯多德：地心說。提出乙太概念說明需要力才能使物體運動。

歐幾里德：幾何原本。

阿基米德：槓桿。浮力。

張衡：渾天儀。

托勒密：以大小輪解釋地心說。

花剌子密：阿拉伯數字。

費波那契：阿拉伯數字引入歐洲。

沈括：夢溪筆談。

畢升：活字印刷。

馬可波羅：來中國，回去後寫書。

古騰堡：鉛活字印刷。

**哥白尼：1543 => 明**。日心說(並非首創，但經詳細計算體系，出版天體運行論)。西方科學史上第一個分水嶺。

布魯諾：拓展哥白尼理論，主張無限宇宙論。1600被焚死。

**加利略：1564-1642 => 明**。現代科學之父(愛因斯坦說的)。鐘擺(據說大學，但有人考證說那時候大學還裝沒裝鐘擺)。維尼斯，改良望遠鏡。觀察月亮。觀察土星->四個衛星。出書 The starry massege。慣性->一個運動物體，在沒摩擦力的表面，恆等速度運動。太陽黑子。出書 The discourse concerning two new science。
